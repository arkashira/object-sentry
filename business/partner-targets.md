# Partner Targets
## Introduction
Object-Sentry aims to integrate with key SaaS and API platforms to enhance its computer vision capabilities and expand its reach. The following partner targets are prioritized based on their potential to add value to Object-Sentry's users, affiliate or revenue-share opportunities, and the effort required for integration.

## Partner Integration Roadmap

1. **Google Cloud Vision API**
	* Free-tier limit: 1,000 units per feature per day
	* Integration effort: Medium (M)
	* Value-add: Enhances image processing and object detection capabilities for users
	* Affiliate/revenue-share: Yes, through Google Cloud Partner Advantage Program
2. **Amazon Rekognition**
	* Free-tier limit: 5,000 images per month
	* Integration effort: Large (L)
	* Value-add: Provides advanced facial analysis and sentiment analysis for users
	* Affiliate/revenue-share: Yes, through Amazon Partner Network
3. **Microsoft Azure Computer Vision**
	* Free-tier limit: 5,000 transactions per month
	* Integration effort: Medium (M)
	* Value-add: Offers advanced image processing and object detection capabilities for users
	* Affiliate/revenue-share: Yes, through Microsoft Azure Partner Program
4. **Clarifai**
	* Free-tier limit: 100,000 operations per month
	* Integration effort: Small (S)
	* Value-add: Provides pre-trained machine learning models for image and video analysis
	* Affiliate/revenue-share: Yes, through Clarifai Partner Program
5. **OpenCV**
	* Free-tier limit: None (open-source)
	* Integration effort: Small (S)
	* Value-add: Offers a wide range of computer vision algorithms and libraries for users
	* Affiliate/revenue-share: No
6. **IBM Watson Visual Recognition**
	* Free-tier limit: 1,000 images per month
	* Integration effort: Medium (M)
	* Value-add: Provides advanced image recognition and classification capabilities for users
	* Affiliate/revenue-share: Yes, through IBM Watson Partner Program
7. **Tencent Cloud Computer Vision**
	* Free-tier limit: 5,000 images per day
	* Integration effort: Large (L)
	* Value-add: Offers advanced image processing and object detection capabilities for users
	* Affiliate/revenue-share: Yes, through Tencent Cloud Partner Program

## Prioritization
The partner targets are prioritized based on the following criteria:

1. Affiliate or revenue-share opportunities
2. Value-add to Object-Sentry's users
3. Integration effort required
4. Free-tier limits and pricing models

By integrating with these partner targets, Object-Sentry can enhance its computer vision capabilities, expand its reach, and generate additional revenue streams through affiliate or revenue-share programs.
