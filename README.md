# Object Sentry

Object Sentry is a Python project that provides a platform for object detection models.

## Features

* List available object detection models
* Select and configure object detection models
* Validate model configurations

## Usage

1. Create an instance of the `ObjectSentry` class.
2. Use the `get_available_models` method to retrieve a list of available models.
3. Use the `select_model` method to select a model.
4. Use the `configure_model` method to configure the selected model.
5. Use the `validate_model_config` method to validate the model configuration.
